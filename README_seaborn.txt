
College Attendance & Performance Analytics System (Seaborn)

Project Description:
This project analyzes college student attendance and academic performance.
It identifies weak and top-performing students, calculates attendance percentages,
and shows relationships between attendance and marks using Seaborn visualizations.
Seaborn provides a modern, aesthetic, and slightly fancy graphical representation.

Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- CSV Files

Project Structure:
DATA/
 - students.csv
 - attendance.csv
 - marks.csv
seabourn.py
README_seaborn.txt
report_seaborn.docx

How to Run:
1. Install required libraries:
   pip install pandas numpy matplotlib seaborn
2. Run the program:
   python seabourn.py
